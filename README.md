# flickr-app

## Install

Git clone the repo
```
git clone https://github.com/chmcphoy/flickr-app
```
Install all dependencies
```
npm install
```

## Deploy (dev server)
```
npm run start
```
Then browse to http://localhost:8080/
## Deploy (for production)
```
npm run build
```
Then open up 'index.html' without a server (tested in Firefox & Chrome).
